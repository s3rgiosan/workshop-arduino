Arduino

O que é o Arduino

  • Plataforma de prototipagem electrónica
  • Artistas, designers, tinkers, engenheiros, ...
  • Criar objectos e ambientes interactivos
  • Open-source
  • Baixo custo
  • Pode operar sem a presença de um computador
  • Expansão "ilimitada" através de shields
  • Linguagem de programação simples
  • Interacção com programas Java, Flash, Processing, ...

Arduino

  • Microcontrolador: ATmega328
  • Tensão de operação: 5V
  • Tensão de input: 7-12V
    (limite: 6-20V)
  • Inputs/Outputs digitais: 14
    (pins 0-13)
  • Inputs analógicos: 6
    (pins 0-5)
  • Outputs analógicos: 6 (PWM)
    (pins 3, 5, 6, 9, 10 e 11)

Arduino

  • Corrente DC por pin de I/O: 40 mA
  • Corrente DC: 50 mA
  • Memória flash: 32KB
    0.5 KB são do bootloader
  • Porta USB
  • Botão para reset
  • LEDs de estado
  • Velocidade de relógio: 16MHz

Electrónica

Lei de Ohm

Relaciona as três grandezas fundamentais da electricidade:

  • Resistência (Ohm)
  • Diferença de potencial (Volt)
  • Intensidade da corrente (Ampere)
  • R = V / I
  • V = R * I
  • I = V / R

Potência eléctrica: P = V * I (Watt)

loop()

  • Executado continuamente

Métodos

  • pinMode(pin, modo) – definir um pin como INPUT ou OUTPUT
  • digitalWrite(pin, valor) – atribuir um valor ao pin (LOW/HIGH)
  • digitalRead(pin) – ler o valor do pin (LOW/HIGH)
  • analogWrite(pin, valor) – atribuir um valor aos pins PWM (0/255)
  • analogRead(pin) – ler o valor do pin analógico (0/1023)
  • delay(ms) – parar o processamento por tempo determinado (msegs)
  • millis() – ler o tempo de execução do Arduino (reset automático ao fim de 5 dias +/-)
  • ...

Analog Input

E se quisermos controlar a intensidade de luz de um LED?

Usamos um pin ADC

  • Converte sinal analógico em digital
  • Arduino tem 6 inputs ADC
  • Conversor tem 10 bits de resolução
    (0-1023)
  • Valores variam entre 0 e 5 volts

Potenciómetro

Mas o input varia entre 0-1023 e o output entre 0-255...

map(value, fromLow, fromHigh, toLow, toHigh)

Potenciómetro

		int sensorPin = 0;
		int ledPin = 9;
		int sensorValue = 0;
		int valMap = 0;

		void setup() {
			pinMode(ledPin, OUTPUT);
		}

		void loop() {
			sensorValue = analogRead(sensorPin); // 0-1023
			valMap = map(sensorValue, 0, 1023, 0, 255);
			analogWrite(ledPin, valMap);
		}

Serial Monitor

Métodos

  • Serial.begin(velocidade) – inicializar a porta Serial
  • Serial.println(valor) – enviar dados para a porta Serial
  • Serial.println(valor, formato) – enviar dados para a porta Serial
  • Serial.read() – ler dados da porta Serial

Arduino -> PC

		int LED = 9;
		int val = 123;

		void setup() {
			pinMode(LED, OUTPUT);
			Serial.begin(9600);
		}

		void loop() {
			Serial.print(val);       // envia ASCII "123"
			Serial.print(val, DEC);  // igual ao de cima
			Serial.print(val, HEX);  // envia ASCII hexadécimal "7B"
			Serial.print(val, BIN);  // envia ASCII binário "01111011"
			Serial.print(val, BYTE); // envia 1 byte
		  
			digitalWrite(LED, HIGH);
			delay(1000);
			digitalWrite(LED, LOW);
			delay(1000);
		}

PC -> Arduino

		int LED = 9;

			    // e.g. quarter note = 1000 / 4, eighth note = 1000/8, etc.

			    tone(8, melody[thisNote],noteDuration);
